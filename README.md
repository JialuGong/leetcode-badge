# Leetcode Badge

> a tool help you show your leetcode ranking | skills | others

### How to use?
- For users
  `http://yourserver:yourport/{your-leetcode-name}/ranking`<br>
  
  Then you will get a badge like this 

  ![badge](./.image/badge.svg)

- For developer
  ```shell
  $ git clone
  # For develop
  $ npm run serve
  ```
